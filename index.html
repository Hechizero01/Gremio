¿Que es HTML?

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language ) es el componente más básico de la Web. Definir el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia.
"Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web. Los enlaces son un aspecto fundamental de la Web. Al subir contenido a Internet y vincularlo a las páginas creadas por otras personas, te conviertes en un participante activo en la Red Informática Mundial.

HTML utiliza "marcas" para etiquetar texto, imágenes y otro contenido para mostrarlo en un navegador Web. Las marcas HTML incluyen "elementos" especiales como: head, title, body, header, footer, article, section, p (parrafo), , div, span, img, aside, audio, canvas, video, ul, ol, li, br, , , , y muchos otros.

Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas", que consiste en el nombre del elemento rodeado por " <" y " >". El nombre de un elemento dentro de una etiqueta no distingue entre mayúsculas y minúsculas. Es decir, se puede escribir en mayúsculas, minúsculas o una mezcla.

¿Que es CSS?

CSS son las siglas en inglés de Cascading Style Sheets, que significa «hojas de esilo en cascada». Es un lenguaje que se usa para estilizar elementos escritos en un lenguaje de marcado como HTML.

La relación entre HTML y CSS es muy fuerte. Dado que HTML es un lenguaje de marcado (es decir, constituye la base de un sitio) y CSS enfatiza el estilo (toda la parte estética de un sitio web), van de la mano.

CSS no es técnicamente una necesidad, pero no querrás tener un sitio que solo tenga HTML, ya que se vería completamente desnudo.

¿Que es JavaScript?

JavaScript es uno de los lenguajes de programación más utilizados. Es el segundo lenguaje de codificación más popular que existe. Claramente, JavaScript es una parte integral de nuestra vida diaria en línea. Pero, ¿qué es exactamente JavaScript y para qué se puede utilizar?

Junto con HTML y CSS, JavaScript es uno de los tres componentes principales de Internet. Cualquier sitio web que visite está obligado a utilizar una combinación de estos tres lenguajes de programación, cada uno con su propio propósito particular. Echemos un vistazo a ellos con más detalle.

Lenguaje de marcado de hipertexto (HTML): este es el lenguaje de programación básico que forma la base de todos los sitios web. Le permite crear contenido, ponerlo en una página y vincular a otras páginas. Hojas de estilo en cascada (CSS): este lenguaje de secuencias de comandos le permite diseñar el contenido de su página web. Es responsable del estilo del texto, los colores, las fuentes, etc. JavaScript (JS): por último, pero no menos importante, JavaScript le permite hacer que su sitio web sea interactivo. Permite que todo el contenido que haya creado con HTML y diseñado con CSS se vuelva más atractivo. Desde botones en los que se puede hacer clic hasta opciones de inicio de sesión, JavaScript es lo que convierte su sitio web de un muro de texto en una experiencia.
